## Offices

Office list manager, where you can add, edit and delete offices. Some animations and transitions were added to make the user experience more comfortable.

### Demo

[**Offices**](https://charming-hotteok-8dc02a.netlify.app)

### Tools

-   Nuxt
-   TailwindCSS
-   Vue
-   Vuelidate
-   VueMask
-   Vuex (Modules)

### Setup

#### Install

```
yarn install
```

#### Run

```
yarn dev
```
